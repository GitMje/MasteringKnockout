Data-Bind Samples

Standard Binding

<input data-bind="value: firstName" /">

Multiple Bindings

<input data-bind="value: firstName, enable: canEdit" /">

Standard Binding to Function

<button data-bind="click: toggleEdit>Toggle Edit</button">

Binding to Function using the model parameter

<ul data-bind="foreach: contacts">
    <li>
    	<span data-bind="text: name"></span>
    	<button data-bind="click: $parent.removeContact">Remove</button>
    </li>
</ul>

Nested Property

<input data-bind="value: location.name" /">

Nested Observable Property

<input data-bind="value: observableLocation().name" /">

Expressions

Old enough to Drink in the U.S.

Function Expressions

<button data-bind="click: function(data) { console.log(data.age())  }"">Log Age</button">
